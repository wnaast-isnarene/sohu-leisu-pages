# sohu-leisu-pages

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容相互独立，不针对任何特定网站或域名。

## 项目简介

这是一个静态 HTML 页面集合仓库。页面以归档形式保存，可独立访问，不依赖后端服务。适用于保存历史页面快照、展示独立功能页面或作为页面素材库。

## 目录说明

仓库主要目录结构如下：

```
├── pages/          # 存放各 HTML 页面文件
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹或文件代表一个独立页面。
- `assets/` 目录用于存放页面公共资源，避免重复。
- 各页面不共享数据，也不依赖外部 API。

## 页面归档说明

- 每个页面均为完整的 HTML 文档，可离线或在线直接打开。
- 页面可能包含样式和脚本，但不会收集用户信息或进行追踪。
- 归档内容保持原始状态，不进行修改或优化，仅保证可正常渲染。

## 维护说明

- 本仓库以存档和展示为目的，不保证页面内容长期更新。
- 如需引用或使用本仓库中的页面，请遵守相关开源许可协议（如有）。
- 欢迎提交 Issue 讨论页面问题，但不保证及时修复。
- 不接受涉及敏感内容或违反法律法规的页面提交。

## 使用方式

1. 克隆或下载本仓库。
2. 使用浏览器直接打开 `pages/` 下的 HTML 文件即可浏览。
3. 也可通过 GitHub Pages 等功能部署在线访问（需自行配置）。
